# HacKit 2026 Design Guide

## Color Scheme
- **Primary Background**: `#fff3b0` (Warm peach/cream color - rgb(255, 243, 176))
- **Text Color**: Dark gray/black (`#2d2d2d` or similar)
- **Accent Color**: Orange (from logo)
- **Secondary**: White with subtle transparency

## Typography
- **Font Family**: "Noto Sans JP", "Noto Sans JP Fallback", sans-serif
- **Logo Style**: Bold, large sans-serif (appears to be a geometric sans-serif)

## Visual Elements
- **Logo**: Orange "Hackit" text with geometric style
- **Background**: Warm cream/peach gradient
- **Design Language**: Modern, minimalist, tech-focused
- **Atmosphere**: Creative, collaborative, energetic

## Key Branding Elements
- Tagline: "繋がる、創る、超えていく。" (Connect. Create. Go Beyond.)
- Event Focus: Cross-project collaboration hackathon
- Target Audience: KIT students, particularly 1st-year students
- Tone: Professional yet approachable, innovative

## Form Design Recommendations
1. Use the warm peach background (#fff3b0) as the primary background
2. Apply "Noto Sans JP" font throughout
3. Use orange accents for buttons and highlights
4. Maintain clean, minimalist design
5. Add visual hierarchy with the warm color scheme
6. Include HacKit branding elements
